Python Backtesting & Monte Carlo Engine

Designed for research purposes only. This tool processes CSV exports from Pine Script and provides institutional-grade risk metrics.

Setup

Place your signal export as signals.csv.

Place your market data as historical_data.csv.

Adjust parameters in config.yaml.

Launch main.ipynb and Run All Cells.

Features

Context Tagging: Automatically detects trend and volatility.

Monte Carlo: Predicts Maximum Drawdown variance.

Plug-and-Play: Zero coding required for the end-user.
